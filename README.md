## 🇧🇷 Sobre o Projeto (Português)
O **NebulaTeen** é um navegador web desenvolvido para derrubar barreiras digitais. Com foco em navegação por voz e interfaces adaptativas, ele busca garantir que a tecnologia seja um espaço inclusivo para todos.

## 🇺🇸 About the Project (English)
**NebulaTeen** is a web browser designed to break down digital barriers. With a focus on voice navigation and adaptive interfaces, it aims to ensure that technology is an inclusive space for everyone.

## 🇪🇸 Sobre el Proyecto (Español)
**NebulaTeen** es un navegador web diseñado para romper barreras digitales. Con un enfoque en la navegación por voz y las interfaces adaptativas, busca garantizar que la tecnología sea un espacio inclusivo para todos.

---

## 🛠 Features / Características
- **Voice Navigation:** Controle o navegador através de comandos de voz.
- **Adaptive UI:** Interface pensada para máxima clareza e acessibilidade.
- **Privacy Focused:** Gestão clara de dados e histórico.

## 🚀 Installation / Instalación
1. Clone the repository: `git clone https://github.com/c6mg8m667g-stack/NebulaTeen.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python main.py`
